# Codex 入门

## 是什么

Codex 是终端 AI 助手，能操控桌面软件、调用 API、写代码。

## 核心能力

- **CUA**：截屏 + 键鼠模拟，操控 Photoshop 等无 API 软件
- **插件系统**：5 个市场，279 个插件可选
- **代理支持**：通过 v2rayN SOCKS5 代理访问远程市场

## 与 Claude Code 分工

| 场景 | 用哪个 |
|------|--------|
| 操控桌面软件 | Codex |
| 写代码改项目 | Claude Code |
| 调 API 服务 | Codex 插件 |
